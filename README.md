# 🤖 بوت ديسكورد احترافي

بوت ديسكورد متكامل يحتوي على:
- **نظام XP والمستويات**
- **نظام الاقتراحات**
- **نظام Giveaway**
- **نظام Broadcast (DM)**
- **تواجد 24/7 في الصوت**

---

## ⚙️ الإعداد السريع

### 1. نسخ ملف الإعدادات
```bash
cp .env.example .env
```

### 2. تعديل ملف `.env`
افتح ملف `.env` وأدخل:
- `DISCORD_TOKEN` — توكن البوت من [Discord Developer Portal](https://discord.com/developers/applications)
- `CLIENT_ID` — معرّف البوت (Application ID)
- `GUILD_ID` — معرّف السيرفر
- `ADMIN_ROLE_IDS` — معرّفات رولات الإدارة (مفصولة بفاصلة)
- معرّفات القنوات المطلوبة

### 3. تثبيت المكتبات
```bash
npm install
```

### 4. نشر الأوامر (يجب عند أول تشغيل)
```bash
npm run deploy
```

### 5. تشغيل البوت
```bash
npm start
```

## 📋 قائمة الأوامر

### 👤 أوامر عامة
| الأمر | الوصف |
|-------|--------|
| `/rank` | عرض مستواك ونقاط XP |
| `/leaderboard` | لوحة المتصدرين |
| `/suggest` | إرسال اقتراح |
| `/help` | قائمة الأوامر |

### 🛡️ أوامر الإدارة
| الأمر | الوصف |
|-------|--------|
| `/giveaway start` | بدء سحب جائزة |
| `/giveaway end` | إنهاء السحب مبكراً |
| `/giveaway reroll` | إعادة سحب الفائزين |
| `/broadcast` | إرسال رسالة لجميع الأعضاء |
| `/suggestion accept` | قبول اقتراح |
| `/suggestion reject` | رفض اقتراح |
| `/setlevel` | تعيين مستوى عضو |
| `/config` | ضبط إعدادات البوت |

## 🏗️ هيكل المشروع
```
discord-bot/
├── src/
│   ├── index.js              # نقطة البداية
│   ├── config.js             # إعدادات البوت
│   ├── commands/
│   │   ├── admin/            # أوامر الإدارة
│   │   └── general/          # الأوامر العامة
│   ├── events/               # أحداث البوت
│   ├── handlers/             # معالجات الأحداث والأوامر
│   └── utils/                # أدوات مساعدة
├── data/                     # قاعدة البيانات (تُنشأ تلقائياً)
├── .env.example              # مثال على ملف الإعدادات
├── package.json
└── README.md
```

## 🔐 الصلاحيات

- **رولات الإدارة** (`ADMIN_ROLE_IDS`): تملك جميع الصلاحيات
- **Administrator Permission**: يُعدّ تلقائياً من الإدارة
- **باقي الأعضاء**: الأوامر العامة فقط

---

## 📝 ملاحظات مهمة

1. **نظام XP**: لكل رسالة 15-25 نقطة مع cooldown 60 ثانية لمنع السبام
2. **Giveaway**: يمكن استخدام `1h`, `30m`, `2d`, `1h30m` لتحديد المدة
3. **Broadcast**: يوجد cooldown 10 دقائق لمنع السبام
4. **قاعدة البيانات**: SQLite — تُحفظ في مجلد `data/`
